## ConsultaCep

App simples feito em Kotlin consumindo API/REST da https://viacep.com.br, com cep que o usuario colocar


- [x] Tratamento de erros da API
- [x] Tratamento com cep menores ou maiores que 8 (padrão)
- [x] Cep inválidos
